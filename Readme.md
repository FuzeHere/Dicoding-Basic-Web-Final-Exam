# 🎮 Genshin Impact Landing Page

A modern and responsive static website about **Genshin Impact**, built using **HTML5** and **CSS3**.  
This project presents a clean overview of the game's background, gameplay mechanics, and profile information in a simple yet visually appealing layout.

---

## ✨ Overview

This project is a **single-page website** that introduces **Genshin Impact** with a modern structure and responsive design.

It includes:

- A **hero / jumbotron section** with title and short introduction
- A **sticky navigation bar** for smooth section access
- A **history section** explaining the development journey of Genshin Impact
- A **gameplay section** covering core combat mechanics
- A **profile sidebar** with quick information
- A **responsive layout** for desktop and mobile screens

---

## 🧩 Tech Stack

- **HTML5**
- **CSS3**
